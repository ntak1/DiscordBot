# Discord Bot to help in SDV

## Features

- Given a character name and the gift key word shows the gifts it likes

```
$$ gift <char name> <loves|likes|neutral|dislikes> <gift name> ?
$$ gift <char name> <loves|likes|neutral|dislikes>
$$ gift <char name>
```
